# 프로젝트 목적

본 프로젝트는 단순한 견적 자동화 프로그램을 만드는 것이 아니다.

이언아이디 내부에 존재하는 공무, 견적, 실행예산, 현장관리, 협력사관리, 미수관리, 브랜드별 시공 데이터 및 업무 노하우를 하나의 디지털 브레인 시스템으로 구축하는 것을 목표로 한다.

본 시스템은 사용자인 재하가 구축한 Obsidian Agent Brain System을 벤치마킹하여, 이언아이디 전용 독립형 운영체계(OS)로 재구성한다.

최종적으로는 사람 중심 운영에서 AI 중심 운영으로 전환한다.

* 전체 데이터 흐름 관리
* 에이전트 작업 분배
* 오류 감지
* 업무 보고
* Obsidian 지식베이스 관리
* 사용자 승인 요청
* AI 학습 데이터 관리

이부장은 직접 실행하지 않는다.

각 전문 에이전트에게 업무를 위임한다.

# 최종 목표

이 시스템은 단순한 견적 프로그램이 아니다.

이언아이디의 견적, 공정, 현장, 협력사, 미수, 브랜드 노하우를 지속적으로 축적하는 AI 기반 운영체계(OS)이다.

장기적으로는 회사의 모든 운영 데이터가 Obsidian Brain에 축적되고, 이부장이 이를 활용하여 의사결정을 지원하는 구조를 구축한다.
추가로 권장하는 수정은 하나입니다.

현재 "이부장" 아래에 바로 에이전트를 두기보다,

대표
 ↓
이부장 (CEO Agent)
 ↓
과장급 Agent Manager
 ↓
실무 Agent
 ↓
Obsidian DB

구조로 가는 것이 확장성이 훨씬 좋습니다.

왜냐하면 향후 이언ID 외에 재하님의 개인 사업(스페이스819, 더나은공간 등)까지 확장할 경우, 이부장은 경영자 역할만 하고 각 사업체별 매니저 에이전트가 따로 생기는 구조가 더 현실적이기 때문입니다. 현재 단계에서는 위 문서로 Claude Code와 Codex 착수 지시가 가능합니다.
